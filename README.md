# Computer-Fundamentals

## Project Showcase - David JUDEA

This repository gathers the key projects developed during my training, covering Web Development, System Administration, and Algorithmic logic.

## 1. Shell'N Splash (System Administration)
An automated Bash script designed to initialize and configure a Linux environment efficiently.

* **Tech Stack:** Bash Scripting.
* **Key Features:**
    * **System Maintenance:** Automates updates, upgrades, and cleaning of packages.
    * **Environment Setup:** Installs essential tools (Git, Emacs, Zsh) and configures Oh-My-Zsh.
    * **User Management:** Creates specific users and groups (`devs`, `admins`) with strict directory permissions.
    * **Sudoers Config:** Automates specific sudo privileges for users to restart services.

## 2. Code is Right (Algorithm)
A logic-based mini-game inspired by "The Price is Right," focusing on conditional loops and user interaction.

* **Tech Stack:** Python (Algorithm logic).
* **Key Features:**
    * Implementation of a "Guess the Number" mechanic.
    * Management of user inputs and range validation.
    * Feedback loop (Higher/Lower) until the correct price is found.

## 3. My Poolfolio (Web Development)
A responsive personal portfolio website designed to present my profile, skills, and projects.

* **Tech Stack:** HTML5, CSS3, Bootstrap 5, JavaScript.
* **Key Features:**
    * Fully responsive layout using the Bootstrap Grid system.
    * Interactive navigation bar with scroll effects.
    * Project gallery using Bootstrap Cards with hover animations.
    * "One Page" architecture with smooth scrolling.
